### RustCheck
[![Rust](https://github.com/logbyjungle/rustcheck/actions/workflows/rust.yml/badge.svg)](https://github.com/logbyjungle/rustcheck/actions/workflows/rust.yml)  
A simple program written in rust that runs `cargo check` whenever your code updates  
You can customize the time for it to run `cargo check` after you've stopped making changes and saved the file by running rustcheck with the time in milliseconds as an argument
